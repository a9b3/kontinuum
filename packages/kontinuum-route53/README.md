# kontinuum-route53

Create resource record set in route53.

## Requirements

[Setup Instructions](https://github.com/esayemm/kontinuum)

*Note: Remember to set nameserver values for the domain*

## Usage

```sh
./script.sh --name <name> --root <root>
```

`--name`: must match s3 bucket

`--root`: root domain
